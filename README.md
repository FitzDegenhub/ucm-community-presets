# UCM Community Presets

Community camera presets for [Ultimate Camera Mod](https://github.com/FitzDegenhub/UltimateCameraMod) (Crimson Desert).

## How it works

UCM v3 can browse and download presets from this repo directly from the app. Click **Community** in the sidebar to see what's available.

## Contributing

Want to share your camera preset?

1. Export your preset from UCM v3 as a `.ucmpreset` file (Export > UCM Preset)
2. Open a Pull Request adding your file to the `presets/` folder
3. Include a good name, description, and your author name in the preset metadata

## Format

Each `.ucmpreset` file is a JSON containing camera settings and the full session XML. UCM reads these directly — no conversion needed.
